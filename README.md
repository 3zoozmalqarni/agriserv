# نظام إدارة القسم البيطري وقسم المختبر في محجر ميناء جدة الإسلامي

نظام إلكتروني متكامل لإدارة قسم المختبر والقسم البيطري في محجر ميناء جدة الإسلامي.

---

## نبذة عن المشروع

نظام شامل يجمع ثلاثة أقسام رئيسية:

### قسم المختبر
- استقبال وإدارة العينات المخبرية
- إدخال واعتماد نتائج الفحوصات
- طباعة التقارير المخبرية
- إدارة المخزون والمواد
- حاسبة التخفيف للمحاليل

### القسم البيطري
- تسجيل الإجراءات البيطرية
- تسجيل إرساليات الحيوانات
- متابعة حالة العينات في المختبر
- عرض نتائج الفحوصات
- تقارير شاملة

### قسم المستفيدين
- تتبع حالة الإرساليات
- متابعة مراحل الفسح (4 مراحل)
- تقييم الخدمات المقدمة
- إدارة حسابات المستفيدين

---

## بيانات تسجيل الدخول

**جميع الأقسام تستخدم نفس البيانات:**

```
اسم المستخدم: admin
كلمة المرور: 123456
```

### اختيار القسم عند تسجيل الدخول:
- قسم المختبر
- القسم البيطري
- قسم المستفيدين

**ملاحظة:** يُنصح بتغيير كلمة المرور من صفحة "إدارة المستخدمين" بعد أول تسجيل دخول.

## التثبيت والاستخدام

### الخطوة 1: تحميل التطبيق

قم بتحميل ملف التثبيت من مجلد `release`:
```
AgriServ LQ System-Setup-1.0.0.exe
```

### الخطوة 2: تثبيت التطبيق

1. افتح ملف التثبيت
2. اختر مكان التثبيت (أو استخدم المكان الافتراضي)
3. اضغط "Install"
4. انتظر حتى يكتمل التثبيت

### الخطوة 3: تشغيل التطبيق

1. افتح التطبيق من أيقونة سطح المكتب
2. أو من قائمة ابدأ → AgriServ LQ System

### الخطوة 4: تسجيل الدخول

1. اختر القسم المطلوب
2. أدخل اسم المستخدم: `admin`
3. أدخل كلمة المرور: `123456`
4. اضغط "تسجيل الدخول"

## البداية السريعة

### لأول مرة:

**قسم المختبر:**
- أضف مواد جديدة في "إدارة المخزون"
- أضف مستخدمين جدد في "إدارة المستخدمين"

**القسم البيطري:**
- سجّل إجراء بيطري جديد
- سجّل إرسالية حيوانات

**قسم المستفيدين:**
- تتبع حالة الإرساليات
- قيّم الخدمات

---

## مراحل سير العمل

```
القسم البيطري → تسجيل إجراء

قسم المختبر → استقبال عينات → إدخال نتائج → اعتماد نتائج

القسم البيطري → عرض النتائج → تسجيل إرسالية

قسم المستفيدين → تتبع المراحل → تقييم الخدمة
```

## صلاحيات المستخدمين

### في كل قسم يوجد:

**1. مدير البرنامج (الحساب الافتراضي)**
- الوصول الكامل لجميع الصفحات
- إدارة المستخدمين
- تعديل وحذف البيانات

**2. مدير المختبر/المحجر**
- إدارة العمليات اليومية
- إدخال وتعديل البيانات
- طباعة التقارير

**3. أخصائي/فني**
- إدخال البيانات
- عرض التقارير
- محدود في التعديل والحذف

**4. مستخدم قراءة فقط**
- عرض البيانات فقط
- لا يمكنه التعديل

## موقع قاعدة البيانات

```
C:\Users\<اسم_المستخدم>\AppData\Roaming\AgriServ LQ System\agriserv-lq.db
```

### للوصول السريع:
اضغط `Windows + R` ثم اكتب: `%APPDATA%\AgriServ LQ System`

## النسخ الاحتياطي

### لعمل نسخة احتياطية:

1. أغلق التطبيق تماماً
2. انتقل إلى المجلد: `%APPDATA%\AgriServ LQ System`
3. انسخ ملف `agriserv-lq.db`
4. احفظه في مكان آمن مع التاريخ

**مثال:** `agriserv-lq-backup-2025-01-19.db`

### لاستعادة النسخة الاحتياطية:

1. أغلق التطبيق
2. اذهب للمجلد
3. احذف الملف القديم
4. انسخ النسخة الاحتياطية
5. غيّر اسمها إلى `agriserv-lq.db`
6. شغّل التطبيق

**مهم جداً:** اعمل نسخة احتياطية بشكل دوري (يومي أو أسبوعي)

## حل المشاكل الشائعة

### 1. التطبيق لا يفتح
- تأكد من إتمام التثبيت بنجاح
- أعد تشغيل الكمبيوتر
- أعد تثبيت التطبيق

### 2. لا أستطيع تسجيل الدخول
- تأكد من البيانات: `admin` / `123456`
- تأكد من اختيار القسم الصحيح
- احذف قاعدة البيانات وشغّل التطبيق من جديد

### 3. البيانات لا تظهر
- تأكد من الصلاحيات
- تحقق من وجود بيانات مسجلة
- راجع القسم الصحيح

### 4. بيانات القاعدة القديمة لا تعمل
إذا كنت ثبّت نسخة سابقة:
1. أغلق التطبيق
2. احذف قاعدة البيانات القديمة من: `%APPDATA%\AgriServ LQ System\agriserv-lq.db`
3. شغّل التطبيق - ستُنشأ قاعدة جديدة تلقائياً

### 5. مشكلة في الطباعة
- تأكد من تثبيت طابعة
- تأكد من وجود بيانات للطباعة
- جرّب الطباعة كـ PDF أولاً

## دليل المستخدم السريع

### قسم المختبر:

**استقبال العينات:**
- اذهب لـ "استقبال العينات"
- ابحث عن رقم الإجراء البيطري
- سجّل العينات المستلمة

**إدخال النتائج:**
- اذهب لـ "إدخال النتائج"
- ابحث عن العينة وأدخل النتائج

**اعتماد النتائج:**
- اذهب لـ "اعتماد النتائج"
- راجع النتائج واعتمدها

**طباعة التقرير:**
- اذهب لـ "طباعة النتائج"
- ابحث عن الإجراء واطبع

### القسم البيطري:

**تسجيل إجراء:**
- اذهب لـ "تسجيل إجراء بيطري"
- أدخل البيانات واحفظ

**تتبع العينات:**
- اذهب لـ "حالة العينات في المختبر"
- ابحث عن رقم الإجراء

**تسجيل إرسالية:**
- اذهب لـ "تسجيل إرسالية حيوانات"
- أدخل البيانات واحفظ

### قسم المستفيدين:

**تتبع إرسالية:**
- اذهب لـ "تتبع حالة الإرساليات"
- أدخل رقم الإجراء والتاريخ
- شاهد المراحل الأربع

**تقييم الخدمة:**
- بعد اكتمال المراحل ستظهر رسالة للتقييم
- أجب على الأسئلة وأرسل التقييم

## للمطورين

### بناء التطبيق من الكود:

```bash
# تثبيت المكتبات
npm install

# بناء التطبيق
npm run build

# بناء نسخة Windows
npm run electron:build:win
```

### التقنيات المستخدمة:
- Frontend: React 18 + TypeScript + Vite
- Styling: Tailwind CSS
- Desktop: Electron 38
- Database: SQLite (better-sqlite3)
- Forms: React Hook Form + Zod
- Reports: jsPDF + html2canvas
